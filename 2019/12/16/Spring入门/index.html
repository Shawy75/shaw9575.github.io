Spring简介

Spring 特点

  1. 轻量级框架: Spring是非侵入性的。-基于Spring开发的应用中对象可以不依赖于Spring的API。
  2. 依赖注入：IOC容器 — 控制反转，DI(Dependency Injection)
  3. AOP面向切面编程
  4. 对事务的支持
  5. 对框架的支持:Spring是一个容器，因为它包含并且管理应用对象的生命周期。
  6. 一站式: 在IOC和AOP的基础上可以整合各种企业应用的开源框架和优秀的第三方类库（实际上Spring自身也提供了展现层的SpringMVC和持久层的SpringJBDC）
